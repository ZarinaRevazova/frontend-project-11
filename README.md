# RSS-reader 

### Description

- web-application for subscribing to RSS feeds by url. You could add sources, view post titles and descriptions, and track read articles in a unified interface.
- [App link](https://frontend-project-11-beta-seven.vercel.app/)

#

### Minimum system requirements

- Node.js 18.0.0 or higher. Check version (in your terminal): `node -v`

#

### Setup the application

Clone this repository: 
`git clone`

Installing: 
`make install`

Launching:
`make develop`

#

### Some RSS-link examples:
- [Lorem RSS](https://lorem-rss.hexlet.app/feed) You can test the API here and monitor real-time updates occurring at various intervals.
- [Lenta.ru](http://lenta.ru/rss/news) Various news.
- [Astro-forecast](http://feeds.feedburner.com/Astrobene).
